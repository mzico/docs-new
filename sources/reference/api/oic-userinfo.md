## API Document

### /oxauth

#### Overview


#### `/oxauth/userinfo`
##### requestUserInfoPost
**POST** `/oxauth/userinfo`

Returns Claims about the authenticated End-User.
The Access Token obtained from an OpenID Connect Authentication Request is sent as a Bearer Token.

###### URL
    http://gluu.org/oxauth/userinfo
