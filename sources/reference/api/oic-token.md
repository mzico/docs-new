## API Document

### /oxauth

#### Overview


#### `/oxauth/token`
##### requestAccessToken
**POST** `/oxauth/token`

To obtain an Access Token, an ID Token, and optionally a Refresh Token, the RP (Client) sends a Token Request to the Token Endpoint to obtain a Token Response
To obtain an Access Token, an ID Token, and optionally a Refresh Token, the RP (Client) sends a Token Request to the Token Endpoint to obtain a Token Response
